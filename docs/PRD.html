1. Product Overview

1.1 Vision

Build an AI memory system that thinks like a human subconscious ‚Äî continuously processing, connecting, and anticipating rather than simply storing and retrieving.

1.2 Problem

Current AI Limitations

  • Stateless ‚Äî Each conversation starts from zero
  • Reactive ‚Äî Only finds info when explicitly asked
  • No insights ‚Äî Cannot connect disparate facts
  • No temporal awareness ‚Äî Cannot track changes over time

1.3 Solution

üé≠ Front-End Agent

  • Fast conversational interface
  • Sub-200ms latency
  • Hot cache for instant retrieval
  • WebSocket streaming

üß† Memory Kernel

  • Async background processing
  • Three-phase loop
  • Activation-based graph
  • Pattern detection
